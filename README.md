# gitgah

# 🤖 ربات فروش دوره‌های آموزشی تلگرام

> ربات فروشگاهی حرفه‌ای برای تلگرام | فروش دوره‌های برنامه‌نویسی و آموزش‌های تخصصی با پرداخت آنلاین

---

## ✨ ویژگی‌های ربات

| بخش | توضیحات |
|-----|---------|
| 🛒 **فروشگاه دوره** | نمایش لیست دوره‌ها با قیمت و توضیحات کامل |
| 💳 **پرداخت آنلاین** | اتصال به درگاه پرداخت زرین‌پال |
| 📚 **دسته‌بندی دوره‌ها** | تفکیک دوره‌ها بر اساس موضوع (پایتون، جنگو، وردپرس و ...) |
| 🎫 **کد تخفیف** | قابلیت اعمال کدهای تخفیف ویژه |
| 👤 **پنل کاربری** | مشاهده دوره‌های خریداری شده |
| 📥 **ارسال خودکار** | ارسال فایل دوره بعد از خرید موفق |
| 🔔 **اعلان مدیریت** | ارسال پیام به ادمین هنگام خرید جدید |
| 🗂️ **ذخیره در JSON** | ذخیره اطلاعات کاربران و خریدها در فایل JSON |

---

## 🛠️ تکنولوژی‌های استفاده شده

| ابزار | توضیحات |
|-------|---------|
| 🐍 **Python** | زبان برنامه‌نویسی اصلی |
| 🤖 **PyTelegramBotAPI** | کتابخانه رسمی و سبک تلگرام (telebot) |
| 🎛️ **Inline & Reply Keyboards** | کیبوردهای شیشه‌ای و دکمه‌های متنی |
| 📁 **JSON** | ذخیره‌سازی اطلاعات کاربران و دوره‌ها |
| 🔐 **python-dotenv** | مدیریت متغیرهای محیطی |

---

## 📁 ساختار پروژه

telegram-course-bot/
│
├── gitgah.py # فایل اصلی ربات
├── courses.json # اطلاعات دوره‌ها (نام، قیمت، توضیحات)
├── users.json # اطلاعات کاربران (آیدی، نام، خریدها)
├── requirements.txt # کتابخانه‌های مورد نیاز
├── .env.example # متغیرهای محیطی نمونه
├── .gitignore # فایل‌های نادیده گرفته شده
├── README.md # توضیحات پروژه
│
├── data/ # پوشه اطلاعات
│ └── purchases.json # اطلاعات خریدها (کاربر، دوره، وضعیت، رسید)
│
└── receipts/ # پوشه عکس رسیدها
└── (عکس‌های رسید ذخیره می‌شوند)
